## What hidden file/folder demonstrated
If you use `ls` command, the hidden file/folder will not show up at all but if you want them to be in the output, you have to put `-a` option so that it also shows hidden files/folders.

## What command output did you redirect into a file?
Command that uses `>`, for example `ls > files.txt`. So what this does is that it instead output the result for `ls` in the command interface, it directly writes the output into `files.txt`.

## Why is guessing in the terminal risky?
Because you may not know what you are actually trying to do on the command interface. Commands like `rm` and `>` do not require your verification or warning to execute, and the commands overwrite or remove the file, which is significant.
